## 💡 What am I doing here?

I like to build applications, internal tools, and developer-facing open-source libraries. I like exploring and learning how well-known visualization libraries work and have built my own catering to the needs of private entities as well as some open-source tools/plugins/component libraries. 

I am currently working on PyNote which is a snappy, zero-setup Python Notebook environment focused on presentation and UX. It combines many of the things I have learned from building the aformentioned apps and tools. I have been solo-developing it the last year and a half and I am aiming for the first official release to be available end of 2026.
